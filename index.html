CloverOS - GNU/Linux

A minimalist Gentoo based operating system designed for performance and ease of use.

Why CloverOS?

Respects your freedom.

CloverOS is one of the few operating systems that is both easy to install and respects your freedom out of the box.

Gentoo GNU/Linux without the complexity.

CloverOS is an as default as possible Gentoo install with all configuration done in /etc/portage/make.conf and the user's home folder.

Minimalist, bloat free configuration.

A CloverOS install gets a desktop up and running with no hassle. Its desktop uses Xorg, FVWM, OpenRC, dhcpcd, xbindkeys, wpa_supplicant, ALSA and Infinality.

No SystemD/avahi/pulseaudio

We've removed systemd/avahi/pulseaudio, making it 100% Poettering-free.

A brief overview of CloverOS

CloverOS is a performance tuned minimal Gentoo GNU/Linux installation intended to be easy to install and use out of the box. Packages are built with Ofast/LTO/Graphite when possible. It uses FVWM/dhcpcd/wpa_supplicant by default and gives you the option to install and load any DE/WM on boot via ~/.bash_profile

This configuration is for you if you are looking for an easy distro that does not include Systemd or Pulseaudio.
